# 🤖 AI Automation Starter Kit

> Stop building from scratch. Start automating in minutes.

The complete toolkit for building AI-powered automations — from simple workflows to autonomous agents.

---

## What's Inside

### 📁 `/templates`
Ready-to-use workflow templates for common automation patterns:
- Email processing & auto-response
- Document extraction & summarization
- Lead qualification & routing
- Content repurposing pipeline
- Meeting summarization
- Browser automation & scraping
- Multi-agent orchestration
- Autonomous research agent
- **AI Security Auditor** ⭐ *NEW - detect prompt injection, API key exposure, data leakage*

### 📁 `/prompts`
Battle-tested GPT prompts optimized for automation:
- Classification & routing prompts
- Extraction & parsing prompts
- Generation & writing prompts
- Analysis & scoring prompts
- Multi-step reasoning chains

### 📁 `/scripts`
Production-ready code snippets:
- Python utilities for API integrations
- JavaScript/Node automation helpers
- Webhook handlers & processors
- Error handling patterns
- Rate limiting & retry logic

### 📁 `/docs`
Implementation guides:
- Architecture patterns for AI agents
- Best practices for prompt engineering
- Scaling automations guide
- Cost optimization strategies
- Security considerations

---

## Quick Start

1. **Pick a template** from `/templates` that matches your use case
2. **Customize the prompts** in `/prompts` for your domain
3. **Deploy** using the scripts in `/scripts`
4. **Scale** using patterns from `/docs`
